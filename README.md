# Site-vitrine-interactif
Site web multi-pages conçu de A à Z en HTML, CSS et JavaScript vanilla, sans framework. Le projet intègre une navigation responsive adaptée à trois formats d'écran, un système de double thème dynamique, une carte interactive, un quiz fonctionnel et une galerie multimédia. L'ensemble a été réalisé dans le cadre d'un projet universitaire (UTT)
